## Description

 TypeScript Levels Application repository.

## Installation

```bash
$ npm install
```

## Setting up the app

```bash
# run migration
$ npx sequelize-cli db:migrate 

# seed  database
$ npx sequelize-cli db:seed:all

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

```
